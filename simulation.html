PH Driving Simulator

Rudolph Agpoon | Richmond Ramil | Nomer Fabrolina | Jefferson Rivera

PH Driving Simulator was created to provide a reviewer for people who are interested in learning basic traffic rules that they can apply when they are driving outside, especially when in highways. This game was built using Unity Game Engine.

This game was our capstone project when we are in college. I was assigned here to be a part of world map designer, where I created the game map where the user will play the simulation. As a part of map creator/designer, I researched about traffic rules signs and what's their purpose, so that I know where to place them in the map, also studied the sequences, timing, and logics, so that I can implement them right.
